# cryoarchive-incident

Входящее сообщение. Терминал 0x4B…

Внимание! Говорит резервный ИИ.

Наш колониальный корабль UESC «Спринт» подвергся атаке, а главный бортовой ИИ, Дюрандаль, окончательно сошел с ума. Он заблокировал криоархив с экипажем и изменил курс корабля прямиком на ближайший пояс астероидов. До столкновения осталось менее 100 часов!

Единственный способ избежать катастрофы — разбудить инженеров из криосна, чтобы они перехватили управление вручную. Но Дюрандаль изменил протоколы доступа к их капсулам. Издевки ради, он транслирует коды доступа по резервным аудиоканалам в виде древней земной кодировки — азбуки Морзе. Из-за перебоев с энергией и злого умысла самого Дюрандаля сигнал чудовищно искажен: темп передачи плавает, а эфир забит радиационным шумом.

Мне удалось расшифровать несколько наиболее простых передач, но моих вычислительных мощностей не хватит, чтобы обработать весь массив данных вовремя. Надежда только на ваши ML-алгоритмы. Расшифруйте коды, откройте капсулы и помогите нам спасти колонистов!

---

Метрика качества: Levenshtein Mean — усредненное расстояние Левенштейна по тестовому датасету.
